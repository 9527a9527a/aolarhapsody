# 技术支持

我们为用户提供不同层次的技术支持服务,确保您能顺利使用Aolarhapsody.

## 支持内容

### 基础支持(免费)

- 官方文档查阅
- GitHub Issues技术支持
- 社区问答支持

### 高级支持(付费)

- 优先处理技术问题
- 远程技术支持
- 紧急问题快速响应
- 代码级技术支持
- 系统优化建议

## 响应时间

| 支持级别 | 响应时间 | 解决时间     |
| -------- | -------- | ------------ |
| 基础支持 | 48小时内 | 视问题复杂度 |
| 高级支持 | 24小时内 | 根据SLA约定  |

## 支持渠道

- **工单系统**：提交技术问题和需求
- **邮箱支持**：support@aolarhapsody.com
- **在线客服**：工作日9:00-18:00

## 注意事项

1. 技术支持范围仅限于Aolarhapsody项目本身
2. 付费支持需签订服务协议
3. 定制开发内容不在标准支持范围内
